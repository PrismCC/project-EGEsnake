# EGEsnake

start on 2023.3.17

completed on 2023.3.31

使用第三方图形库: EGE 20.08

C++标准: C++20

使用IDE: clion

操作系统: windows

---

#### 操作方法

+ 打开EGEsnake.exe进入游戏

+ 依次输入地图名称, 玩家个数, 总玩家数, 以及颜色选择

+ 按键:

  >W S A D :  玩家一
  >
  >I K J L :     玩家二
  >
  >↑↓←→ :    玩家三
  >
  >Esc:           退出游戏
  >
  >R:               回到标题界面
